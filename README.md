# 🚀 3D-MODEL-SANDBOX (Agentic Sandbox)

**A Collaborative Simulation Engine by Saad & Aurang**

Welcome to the **3D-MODEL-SANDBOX**, a state-of-the-art interactive 3D environment designed to simulate and analyze LLM-driven agent architectures. Co-created by **Saad** and **Aurang**, this project merges advanced React Three Fiber visuals with intricate game theory algorithms. Watch as autonomous AI models intelligently navigate dynamic, spatial mazes, making real-time tactical decisions in complex Hunter vs. Prey scenarios.

---

## ✨ Features

- **🧠 LLM-Driven Agents**: Connect to various agentic models via the Groq API and watch them navigate, strategize, and hunt.
- **🎮 Multiple Game Modes**: Choose between Classic, Swarm, and Faction modes to customize the simulation's complexity.
- **👁️ Stunning 3D Visuals**: Built with React Three Fiber, featuring smooth animations, dynamic lighting, and immersive environments.
- **📊 Live Analytics**: Real-time distance tracking, performance metrics, and a dynamic scoreboard to measure the agents' success.
- **⚙️ Deep Customization**: Set game speed, max turns, movement distances, and customize the models assigned to different agents dynamically.

---

## 🛠️ Technology Stack

- **Frontend Front**: React 19, Vite
- **3D Rendering**: Three.js, React Three Fiber, Drei
- **State Management**: Zustand
- **Styling**: Custom modern CSS with Glassmorphism UI elements
- **AI Integration**: Groq API
